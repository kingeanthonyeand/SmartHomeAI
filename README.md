# SmartHomeAI

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## Overview

SmartHomeAI is an advanced home automation platform that leverages artificial intelligence to optimize and simplify smart device management. It learns your preferences, automates tasks, and helps reduce energy consumption for a smarter, greener home.

## Features

- AI-driven device scheduling and automation
- Energy usage analytics and recommendations
- Voice command integration with popular assistants
- Secure remote access via mobile and web apps
- Customizable rules and scenarios

## Tech Stack

- **Backend:** Python, Flask, TensorFlow
- **Frontend:** React, Redux
- **Database:** PostgreSQL
- **Mobile:** React Native
- **AI/ML:** TensorFlow with custom models

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kingeanthonyeand/SmartHomeAI.git
   cd SmartHomeAI
   ```
2. Install backend dependencies:
   ```bash
   pip install -r backend/requirements.txt
   ```
3. Install frontend dependencies:
   ```bash
   cd frontend
   npm install
   ```
4. Setup the database and run migrations.
5. Start backend server:
   ```bash
   python backend/app.py
   ```
6. Start frontend development server:
   ```bash
   cd frontend
   npm start
   ```

## Usage

- Access the web interface at `http://localhost:3000`
- Configure your smart devices and preferences
- Enable AI automation and monitor energy usage
- Use mobile app for remote access
